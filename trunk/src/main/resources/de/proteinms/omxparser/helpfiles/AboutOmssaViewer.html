About OMSSA Viewer

OMMSA Viewer has been developed by Harald Barsnes (PhD student at University of Bergen, Norway) under the guidance of Prof. Dr. Lennart Martens (UGent / VIB). It is a simple, lightweight viewer for OMSSA omx files. It is a part of the OMSSA Parser project.

Project home page: http://omssa-parser.googlecode.com/

License:

OMSSA Parser: Apache License 2.0

The platform independent graphical user interface design is based on JGoodies and has a separate license.

For licenses regarding Java see www.sun.com.

Help:
